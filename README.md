# MiniC_Compiler

A lightweight implementation of a Mini C Compiler developed as part of the Compiler Design project.
This project demonstrates the complete compilation pipeline — from lexical analysis to intermediate code generation — for a subset of the C programming language.
The project simulates how modern compilers such as GCC and Clang translate high-level source code into machine-understandable instructions. It provides a hands-on understanding of compiler design concepts including lexical analysis, syntax analysis, semantic analysis, and intermediate code generation.
This compiler begins with lexical analysis, where the source code is scanned and broken into meaningful tokens such as keywords, identifiers, operators, and constants. The syntax analysis phase validates the structure of the program using grammar rules and constructs a parse tree or abstract syntax tree (AST).
Inspired by how real-world compilers like GCC and Clang process programs internally.

📌 Project Overview

The Mini C Compiler translates a simplified C program into intermediate or target-level code by implementing the fundamental phases of a compiler:
Lexical Analysis
Syntax Analysis
Semantic Analysis
Intermediate Code Generation
(Optional) Code Optimization
Target Code Generation
